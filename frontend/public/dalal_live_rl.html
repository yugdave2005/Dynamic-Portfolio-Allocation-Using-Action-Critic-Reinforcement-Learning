How the Agent Learns:
The Actor-Critic network starts with zero knowledge (random weights). The Actor outputs portfolio allocations via softmax. The Critic estimates the future value. After each simulated year (252 days), the agent computes the reward: R_t = PortfolioReturn - (Cost * Turnover). Positive surprises increase the probability of those weights; negative surprises decrease them. You will watch this optimization happen live.
